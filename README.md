# 🗺️ Wiggle

Wiggle is a spatial, walk-around social media platform that transforms traditional content feeds into an interactive, explorable world. Instead of endlessly scrolling down a static timeline, users navigate a digital canvas to discover, interact with, and contribute to community boards.

## 🚀 Key Features

* **Spatial Navigation:** Walk around an open canvas to discover user-curated content hubs and boards.
* **Interactive Boards:** Walk up to any board to dive into it, read full posts, and explore media.
* **Live Refresh:** Instantly reload posts within any board to fetch the latest updates and community content.
* **Social Engagement:** Like posts, leave comments, and follow your favorite creators directly from the board interface.
* **Immersive Experience:** A seamless blend of spatial exploration and modern social media connectivity.
